### 刷赞方式

这种方式只要是刷无需登录且校验IP地址的活动，也就是，不同的IP地址的人就可以继续点赞，所以就需要寻找代理IP，然后抓取到点赞接口，利用代理IP去请求点赞接口即可达到刷赞的目的，具体的活动的接口接口的相关地址以及请求参数，可以利用`fildder`工具去抓取，然后模拟请求即可。
