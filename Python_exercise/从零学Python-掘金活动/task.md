练习实现思路提示：
1. 沸点数据的请求链接，可以通过F12的network查找
2. 通过Requests的get方法请求沸点url链接
3. 然后用json()方法获取请求到的的内容，是一个字典类型的数据

    2. 在for循环外用写文件的模式打开一个文件对象，然后持续写入
    3. 写完文件最后都要记得关闭文件对象，或者使用with……的语法
